A modern, intuitive project management application built with React and Tailwind CSS.

## Features

- Create and manage multiple projects
- Track tasks with priorities and due dates
- Assign tasks to team members
- Filter projects by status
- Responsive design
- Local data persistence

## Technologies Used

- React 18
- Tailwind CSS
- Lucide React (icons)
- LocalStorage API

## Installation

# Clone repository
git clone https://github.com/ARJUN151503/project-management-system.git

# Navigate to directory
cd project-management-system

# Install dependencies
npm install

# Start development server
npm start

## Usage

1. **Create Projects**: Click the "+" button in the Projects section
2. **Add Tasks**: Select a project and click "+" in Tasks section
3. **Filter**: Use the status dropdown to filter projects
4. **Edit/Delete**: Use the edit and delete icons on each item

## Build for Production

npm run build
